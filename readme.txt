== Description ==

This plugin shows the path to the page or post the user is currently surfing, so he will not get lost in the navigation structure that easily and can navigate quickly.

I suppose this plugin is most useful if you use Wordpress not only as blog, but also as a CMS.

To see this plugin in action visit [my homepage](http://www.hyper-world.de/en/ "hyper-world"). The part below the title image, left from the language selection and search field is generated by this plugin.

== Installation ==

1. Upload `pathbar.php` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the `Plugins` menu in WordPress.
1. Adjust the plugins settings in the `Settings/Pathbar` menu.
1. Place `<?php the_pathbar(); ?>` in your templates at the position where the pathbar should be shown.
